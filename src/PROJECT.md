# Pushti Thala

Premium healthy food delivery web application built with React.

## Tech Stack

- React
- React Router
- Context API
- LocalStorage
- Vite

## Features

- Authentication
- Food Menu
- Search & Category Filters
- Cart Drawer
- Checkout System
- Coupon Engine
- Order Tracking
- Profile Dashboard
- Dark / Light Mode

## Design System

- Glassmorphism UI
- Green Gradient Branding
- Premium Card Components
- Responsive Layout
- Dark Theme First
